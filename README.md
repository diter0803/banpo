# 半坡万象

基于当前文件夹内半坡纹样素材制作的本地互动网站。

## 本地启动方式

双击 `启动网站.bat`。浏览器会自动打开：

`http://127.0.0.1:4173/`

保持弹出的终端窗口开启；关闭窗口即可停止网站。

## GitHub Pages 发布

本项目使用独立的 `docs` 目录作为 GitHub Pages 发布包，本地部署文件不受影响。

重新生成发布包：

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\prepare-github-pages.ps1
```

然后在 GitHub 仓库 `Settings > Pages` 中选择：

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/docs`

详细步骤见 `GITHUB_PAGES.md`。

## 功能

- **五步纹样生成器**：先选择环形、放射、对角或中央骨架，再添加鱼、人面、鹿、蛙等生物母题，填入三角、波折、网格、圆点等几何带，最后加边框并切换红陶、拓片或大地配色。
- **参数化纹样积木**：支持鱼纹抽象度、人面眼口替换、鹿纹完整/鹿角模式、蛙纹屈曲度、几何重复密度与 AB 交替、边框粗细。
- **创作工具**：撤销、重做、自动骨架排版、关键词灵感生成、文化符码解读、浏览器保存、连续图案导出和陶器应用预览。
- **背景音乐控制**：内置背景音乐，可在页面右下角自主打开、关闭并调节音量。
- **自由造形**：使用鼠标或触控绘制形状，或选择游鱼、涡旋和陶钵模板。纹样粒子会规律明灭并轻微漂移。
- **以纹观相**：允许摄像头后识别微笑、惊讶与眨眼，用随机纹样粒子实时组成对应表情。

照片与摄像头画面不会上传或保存。首次启用表情识别时，需要联网读取 MediaPipe 官方运行组件和模型。
